# Personal Portfolio

Welcome to my personal portfolio! This project showcases my professional journey, skills, and interests through a modern and innovative web interface. Below is an overview of the structure and features of the portfolio website.

## Project Structure

```
personal-portfolio
├── src
│   ├── index.html               # Main entry point for the portfolio website
│   ├── styles                   # Contains CSS files for styling
│   │   ├── main.css             # Primary styles for layout and typography
│   │   └── theme.css            # Theme styles for a techie look
│   ├── scripts                  # JavaScript files for interactivity
│   │   └── main.js              # Main script for dynamic content
│   ├── components               # Reusable HTML components
│   │   ├── header.html          # Header component with navigation
│   │   ├── footer.html          # Footer component with copyright info
│   │   └── card.html            # Card component for articles/projects
│   ├── pages                    # Contains different pages of the portfolio
│   │   ├── home                 # Home page
│   │   │   └── index.html       # Content for the home page
│   │   ├── cv                   # CV section
│   │   │   ├── index.html       # Content for the CV page
│   │   │   └── source           # LaTeX source for CV
│   │   │       └── cv.tex       # LaTeX file for CV
│   │   ├── articles              # Articles section
│   │   │   ├── html             # HTML articles
│   │   │   │   └── example-hosted-article.html # Example hosted article
│   │   │   └── latex            # LaTeX articles
│   │   │       ├── curiosity-1.tex # LaTeX file for curiosity article 1
│   │   │       └── curiosity-2.tex # LaTeX file for curiosity article 2
│   │   └── about                # About page
│   │       └── index.html       # Content for the about page
│   └── data                     # Data files for articles
│       └── articles.json        # Metadata for articles
├── public                       # Public assets
│   └── favicon.ico              # Favicon for the website
├── tools                        # Tools for building and processing
│   ├── build                    # Build configuration
│   │   └── build-config.js      # Configuration settings for building
│   └── latex                    # LaTeX build tools
│       └── build-latex.sh       # Script for building LaTeX documents
├── .gitignore                   # Git ignore file
├── package.json                 # NPM configuration file
└── README.md                    # Project documentation
```

## Features

- **Professional CV**: A dedicated section to showcase my curriculum vitae, available in both HTML and PDF formats.
- **Articles Section**: A collection of articles, including hosted HTML articles and LaTeX articles on various curiosities.
- **Responsive Design**: The website is designed to be fully responsive, ensuring a great user experience on all devices.
- **Techie Theme**: The styling incorporates a modern and innovative theme, making use of vibrant colors and sleek layouts.

## Getting Started

To get started with this project, clone the repository and open the `src/index.html` file in your web browser. You can customize the content in the various sections to reflect your own career and interests.
